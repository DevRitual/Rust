# Rust Learning Journey

This repository contains examples and exercises for learning Rust.

## Projects & Files

- `src/main.rs`: Exploring basic ownership concepts, move semantics, and primitive types.
- `helloworld.rs`: Basic "Hello World" with vector manipulation exercises.
- `vector.rs`: A playground for working with Rust vectors.

## How to Run

### Main Project
To run the main cargo project:
```bash
cargo run
```

### Standalone Files
To run the standalone Rust files:
```bash
rustc helloworld.rs
./helloworld
```

## Concepts Covered
- **Ownership & Borrowing**: Understanding how Rust manages memory without a garbage collector.
- **Vectors**: Using `Vec<T>` for dynamic arrays.
- **Primitive Types**: Standard scalar types in Rust.
